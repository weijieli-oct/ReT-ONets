# Dataset Download and Usage

The dataset for this project is available from the GitHub Release page:

[https://github.com/weijieli-oct/ReT-ONets/releases/tag/data](https://github.com/weijieli-oct/ReT-ONets/releases/tag/data)

## How to Use the Dataset

1. Open the link above and download the dataset archive from the Release assets.

2. Extract the downloaded archive into the root directory of“ReT-ONets”code repository.

3. After extracting the dataset, run the training scripts following the project workflow.

## Notes

- If the extracted archive contains an extra top-level folder, move the `data` folder into the repository root directory.
- Make sure that the data paths in the training scripts match the extracted dataset location.
- It is recommended to open the repository root directory in PyCharm and run the corresponding training scripts directly.
